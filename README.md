## Project Description
LearnTrack is a console-based Student and Course Management System built using **Core Java**.
It allows administrators to manage students, courses, and enrollments using a menu-driven
interface.

This project focuses on strengthening Java fundamentals such as:
- OOP concepts (encapsulation, inheritance, polymorphism)
- Collections (`ArrayList`)
- Exception handling
- Clean and modular code structure

All data is stored **in-memory** using collections (no database).

---

## How to Compile and Run

### Prerequisites
- Java JDK 8 or above installed
- Command Prompt / Terminal or IDE (IntelliJ, Eclipse)

### Steps
1. Navigate to the `src` directory:
```bash
cd src
```
2.Compile the project
```bash
javac com/airtribe/learntrack/Main.java
```
Run the application:
```bash
java com.airtribe.learntrack.Main
```
